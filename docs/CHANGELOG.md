# Changelog

Tutte le modifiche rilevanti a questo progetto saranno documentate in questo file.
Il formato è basato su [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2024-01-20
### Aggiunto
- Versione iniziale del plugin.
- Gestione tabelle custom per Movimenti, Conti, Raccolte e Anagrafiche.
- Sistema di importazione CSV (Generico, PayPal, Satispay).
- Generazione Report Modello D e Report Raccolte Fondi.
- Integrazione Select2 per la ricerca anagrafiche.
- Sistema di validazione per integrità dati (can_delete).
