# Roadmap del Progetto

## Versione 1.1.0 (In sviluppo)
- **Hooks System:** Introduzione di filtri per permettere a plugin terzi di aggiungere provider CSV personalizzati.
- **Dashboard Widget:** Grafico rapido entrate/uscite nella bacheca di WordPress.
- **Allegati Multipli:** Supporto a più documenti per singolo movimento (attualmente limitato a 1 tramite tassonomia).

## Versione 1.2.0 (Pianificato)
- **Regole di Importazione:** Attivazione della tabella `terzoconto_regole` per categorizzare automaticamente i movimenti importati basandosi su parole chiave.
- **Export AE:** Implementazione della tabella `terzoconto_comunicazioni_ae` per la generazione dei file per l'Agenzia delle Entrate (erogazioni liberali).
