== Description ==

TerzoConto is a WordPress plugin designed to help Italian Third Sector Organizations (ETS) manage accounting and financial data.

The plugin supports structured data import (CSV), validation, and administrative workflows, simplifying compliance and bookkeeping.

Main features:

* CSV data import
* Data validation
* Administrative interface
* Modular architecture (services and repositories)
* Extensible structure for future integrations

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin from the WordPress Plugins menu
3. Access the plugin section in the admin panel

== Frequently Asked Questions ==

= Which file formats are supported? =
Currently CSV files are supported.

= Is it compatible with other plugins? =
The plugin follows WordPress best practices, but compatibility is not guaranteed.

= Are data validated? =
Yes, imported data are validated according to predefined rules.

== Description ==

Terzo Conto è un plugin WordPress progettato per supportare la gestione e l'elaborazione delle erogazioni liberali degli enti del Terzo Settore.

Il plugin consente di importare, validare e gestire dati strutturati (es. CSV) secondo specifiche tecniche definite, facilitando i processi amministrativi e di controllo.

Funzionalità principali:

* Importazione dati da file CSV
* Validazione dei dati secondo regole definite
* Gestione amministrativa tramite interfaccia dedicata
* Struttura modulare basata su servizi e repository
* Estendibilità per integrazioni future

== Installation ==

1. Caricare la cartella del plugin nella directory `/wp-content/plugins/`
2. Attivare il plugin dal menu 'Plugin' in WordPress
3. Accedere alla sezione dedicata nel pannello di amministrazione

== Frequently Asked Questions ==

= Quali formati di file sono supportati? =
Attualmente il plugin supporta l'importazione di file CSV.

= È compatibile con altri plugin? =
Il plugin è progettato seguendo le best practice WordPress per minimizzare conflitti, ma non sono garantite compatibilità specifiche.

= I dati vengono validati? =
Sì, il plugin applica controlli sui dati importati secondo regole predefinite.
